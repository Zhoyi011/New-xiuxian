【如何游玩】
1.下载全部文件( index.html, script.js, style.css)。
2.保存在同个文件夹。
3.双击index.html (如果你要高级点，就打开cmd，写cd "（文件夹位置）"，Enter，然后再写index.html。
4.本人已开启了网页版，将自动更新最新版本
  https://new-xiuxian.vercel.app/

【注意事项】
1.请勿搬运！本人已经做过标记，所以请勿搬运！
2.只有用电脑才能玩。（Windows可以，Mac、Linux应该也可以吧。）

【特殊玩法】
1.可到script.js 更改境界需求、功法所需灵气等等....本人已经标记所有可更改的地方。
2.可按F12，在console填写gameState.(你需要的东西*注：qi=灵气，exp=修为，spiritstones=灵石) =(数量））例子：gameState.qi = 1000 等于 灵气*变成*1000
3.设置里有一些设置可以去看看哦。
